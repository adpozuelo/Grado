Los ficheros de código fuente .java se encuentran en la siguiente carpeta:

Los ficheros generados por javadoc se encuentran en la siguiente carpeta:

./doc -> todos los ficheros javadoc.

Los ficheros .jar compilados para comprobar el output son los siguientes:

./jar/POO_PRA2.jar -> el fichero del ejercicio.
./jar/*.txt -> inputs & outputs.

El proyecto creado con NetBeans se encuentra en la siguiente ubicación:

./netBeansProject/POO_PRA2.tgz -> el proyecto del ejercicio.
